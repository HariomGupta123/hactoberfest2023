Instructions

Thank you for contributing to Hacktoberfest! To add your name and GitHub profile to the list of contributors, follow these steps:

  1. Star this Repository
  2. Fork this repository.
  3. Edit the Index.html file.
  4. Add your name and GitHub profile URL to the list.
  5. Create a pull request.
